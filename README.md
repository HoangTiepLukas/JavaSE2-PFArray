## Java SE II
This project is structured using 2 main classes :
- **PFArray**: handle arrays operations
- **Main**: provides console menu, interactions

### Functions:
- Create Multiple Arrays
- Create random array
- Create array manually
- Print arrays 
- Find max and min number in array
- Calculate sum of array
- Generate random numbers in an array
- Add number to array
- Delete number from array
- Delete array

### Testing
- **JUNIT 5** for testing
